Présentation

Par groupe de 4 développeurs, il nous a été demandé de réaliser une replique du célèbre jeu de blocs Minecraft, en 2D !

En prenant soin de coder le plus proprement possible, grâce au MVC, nous avons pû implémenter les fonctionnalitées suivantes :

  • Génerer une carte aléatoire, contenant un arbre et un terril.
  • Se déplacer sur la carte sans déclancher d'exception quand on y sort.
  • Miner et stocker dans un inventaire les blocs de la carte.
  • Fabriquer n'importe quel objet, bloc, outil avec une combinaison de craft.
  • Faire fondre des minerais dans un four.
  • L'apparition de créatures qui se déplacent aléatoirement ou qui suivent le joueur

    • Pour réussir à bien ce projet, nous avons utilisé la méthode agile SCRUM. Nous avons donc pu nous organiser en équipe et nous répartir les tâches en fonction de nos compétences.

    Outils

    Java

    Entièrement codé sous IntelliJ IDEA avec le moteur de production Gradle.

    MVC

    Architecture MVC (Modèle-Vue-Contrôleur) pour une meilleure organisation du code.

    JavaFX

    JavaFX est un framework et une bibliothèque d'interface utilisateur issue du projet OpenJFX, qui permet aux développeurs Java de créer une interface graphique. Il utilise des fichiers FXML pour la mise en page et des fichiers CSS pour la mise en forme. Pour la fond, nous utilisons des images que nous disposons dans des cellules.

    GitLab

    Utilisation de GitLab pour le versionning du projet. Nous avons pu créer des branches pour chaque fonctionnalité et les fusionner avec la branche develop une fois terminé. Chaque fonctionnalitée a un ticket associé dans l'Issue Boards et attribué à un membre de l'équipe.
