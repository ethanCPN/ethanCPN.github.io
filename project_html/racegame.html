Présentation

Ce projet personnel a été réalisé entièrement en Python sur PyCharm. Il s'agit d'un jeu de courses de voitures qui demande à l'utilisateur de faire les bons choix pour gagner les duels.

Pour avoir un jeu réaliste et varié, j'ai récupéré les données des voitures du jeu de course Forza Horizon 5 sur le wiki de Fandom. J'ai ensuite créé un fichier CSV avec les données qui m'intéressaient, puis je les récupère séparément pour chaque voiture. En effet, 3 voitures sont tirées aléatoirement pour chaque duel, l'utilisateur doit donc choisir la voiture qui a le plus de chance de gagner contre une voiture tiré elle aussi aléatoirement au début de la manche. Les voitures sont tirés de façon à ce que le joueur ait forcément une chance de gagner, avec des écarts de puissance faible tout de même.

Voici comment se déroule une manche :

  • La voiture de l'adversaire est tirée aléatoirement et ses statistiques sont affichées.
  • Les 3 voitures du joueur sont tirées aléatoirement sans afficher leurs statistiques.
  • Selon les points du joueur, il peut en utiliser 1 pour afficher une information sur une des voitures.
  • Il doit enfin faire son choix et décider sur quel circuit il veut faire la course. Le Run est une ligne droite qui demande de l'acceleration et de la vitesse, tandis que le Circuit demande aussi du freinage et de la maniabilitée.
  • Enfin, la fonction correspondant au circuit choisi calcule les points de run de chaque véhicule et détermine le vainqueur.

    • Pour avoir un code plus lisible et plus facile à modifier, j'ai décider de créer un fichier principal qui gère le déroulement du jeu en appelant les fonctions du fichier functions.py. J'ai pris soin de commenter chaque fonction pour expliquer son fonctionnement.

    Outils

    Python

    Le language Python est trés simple à utiliser et me permet de me concentrer sur la logique du jeu plutôt que sur la syntaxe.

    Panda

    J'utilise le framework Panda pour récupérer les données du fichier CSV et les stocker dans un DataFrame.

    Matpotlib

    Le module Matplotlib permet de créer des graphiques pour afficher la course en temps réel. Cette fonctionnalité est en cours de développement car le language Python ne permet pas de rafraîchir rapidement les graphiques, les requêtes saturent la mémoire et le programme cesse de fonctionner.

    GitLab

    L'tilisation de GitLab me permet de versionner mon code et de le sauvegarder sur un serveur distant. Je peux facilement revenir à une version antérieure en cas de problème et partager mon projet avec d'autres développeurs.
